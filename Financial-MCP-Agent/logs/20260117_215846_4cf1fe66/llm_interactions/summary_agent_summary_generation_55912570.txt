由于数据获取限制，建议关注以下技术指标：

#### 关键技术位识别
- **支撑位**：前期低点、重要均线位置、成交量密集区
- **阻力位**：前期高点、重要技术位、心理整数关口

#### 主要技术指标
- **移动平均线**：关注5日、10日、20日、60日、120日、250日均线
- **MACD指标**：DIF与DEA线关系，MACD柱变化
- **RSI指标**：超买超卖区间识别，背离信号分析

### 技术分析原则
- 趋势确认：判断当前处于上升、下降还是盘整趋势
- 量价配合：价格变动是否有成交量支持
- 多重验证：多个技术指标一致性分析
